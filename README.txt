Project made for Purdue CS352 - Compilers: Principles and Practice, Spring 2014.
Instructor: Gregor Richards
Class Page: https://www.cs.purdue.edu/homes/gkrichar/cs352s2014/

New code was implemented in 6 parts on top of a template provided by the instructor:
1. Lexer (parser/ParserAst.jj)
2. Parser (parser/ParserAst.jj)
3. Compiler to IR (SSACompiler.java)
4. Type Checker (TypeChecker.java)
5. Register Allocator (backend/RegisterAllocator.java)
6. Compiler to MIPS (backend/AsmMips.java, backend/ClassLayout.java)
